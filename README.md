


Reproduces the calculations in our manuscript XYZ. Produces tables a
for latex and png files for the plots.

Follow the procedures in NEW/run_all_tides.sh However, that is nota
script to run - look in it and see th ethings thank can be run from the
command line, and see the R,d code that has to be run from inside Rstudio.

Understand that FIRST you must download the GESLA files. The present code does
expect the GESLA data to be in ~/GESLA4/DATA/

You will have to install the 'uv' package manager - it is described in
the run_all_tides.sh file

You will have to have R and Rstudio installed and likely tinytex

If the codes crash you will see the libraries you need to install,
and then rerun.
